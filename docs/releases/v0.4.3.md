# Conclave AI v0.4.3

Release date: 2026-04-20

**Security-motivated removal of `@conclave-ai/agent-deepseek`.**

## Breaking (narrow)

- **Removed the `@conclave-ai/agent-deepseek` package** from the repo and dropped it from the CLI's agent enum. Config files that previously listed `"deepseek"` in `agents` or `council.domains.*.tier1/tier2` will now fail validation on `conclave init --reconfigure` and on `conclave review`.
- The npm versions `@conclave-ai/agent-deepseek@{0.2.0..0.4.0}` remain published (72-hour unpublish window already passed for early versions) and are marked **deprecated** with a message pointing at this release note. Anyone who pinned them still gets installs; they just see a deprecation warning on `pnpm install`.

## Why

DeepSeek's hosted API sends user code outbound to DeepSeek-controlled infrastructure. Conclave's code-review traffic includes full diffs — often including proprietary logic — and our trust posture doesn't extend to that hop for a hosted agent in v0.4. Dropping the package is narrower than carrying a partially trusted surface.

## Migration

Users with `"deepseek"` in config:

1. Remove `"deepseek"` from `.conclaverc.json`'s `agents` array and from any `council.domains.*.tier1` / `tier2` lists.
2. If you relied on DeepSeek output, the closest replacement is **local Ollama** with a DeepSeek open-weight model:
   ```json
   "agents": ["claude", "openai", "gemini", "ollama"]
   ```
   and `OLLAMA_BASE_URL` pointing at a local/self-hosted Ollama serving `deepseek-r1` or similar. Traffic stays on your machine.

3. Uninstall the now-unused package:
   ```
   pnpm remove @conclave-ai/agent-deepseek
   ```

## What stayed

- **`@conclave-ai/agent-ollama`** — the local-daemon adapter still works and remains the supported way to run DeepSeek-family **open-weight models** on hardware you control.

## Not in this release

- **Paid tier / LLM proxy** — still deferred. Revisit after external install count + retention signal.
- **v0.5 items** (Design agent, i18n, D1 token encryption) — tracked separately.
