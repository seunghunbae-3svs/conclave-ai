# v0.13.7 — `conclave doctor` + design-domain autofix + post-push deploy wait

Released **2026-04-27**.

## TL;DR

Three additions and one fix; ~22 new tests:

1. **`conclave doctor`** — single-command diagnostic of your install. 4 checks (env keys / worker / workflow refs / npm version), one line per check, no LLM cost.
2. **Autofix can now fix design-domain blockers** with a `file` set — Tailwind class swaps, color tokens, aria-* edits all flow through to the worker now (used to be hard-skipped on the assumption visual judgment requires a human).
3. **Post-push deploy wait** — autofix polls Vercel/Netlify check-runs after each push so the next review doesn't see the stale preview.
4. **Hermetic regression tests for v0.13.5-era RCs** — RC#7 (scoped staging), RC#10 (multi-agent dedupe → 1 worker call), RC#13 (`--no-notify` propagation).

See `CHANGELOG.md` for the full v0.13.7 entry.
